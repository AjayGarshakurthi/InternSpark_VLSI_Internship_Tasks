# InternSpark_VLSI_Internship_Tasks🚀 
During this internship, I gained hands-on experience in the complete digital IC design flow, covering RTL design, functional verification, synthesis, gate-level simulation, and static timing analysis.
#Tasks:
1. RTL Design & Functional Verification
• Designed and implemented an 8-bit ALU using Verilog HDL.
• Developed comprehensive testbenches with directed and random test cases.
• Performed functional verification using ModelSim/GTKWave.
• Analyzed simulation waveforms to validate design functionality.

2.Synthesis & Gate-Level Simulation
• Synthesized RTL designs into gate-level netlists using Yosys.
• Generated synthesis reports and technology-mapped netlists.
• Conducted gate-level simulations and compared results with RTL simulations to ensure functional equivalence.

3.Static Timing Analysis (STA)
• Created timing constraints using SDC files.
• Performed setup and hold timing analysis using OpenSTA.
• Generated timing reports and analyzed critical timing paths.
• Studied timing violations and explored optimization techniques for timing closure.

🛠 Tools & Technologies:
Verilog HDL, ModelSim, GTKWave, Yosys, OpenSTA, Linux, TCL, GitHub

📚 Key Skills Acquired:
RTL Design • Functional Verification • Logic Synthesis • Gate-Level Simulation • Static Timing Analysis • Timing Closure • Digital VLSI Design Flow
